## App Architecture

```
"native-base": "2.2.1",
```

## Development

### Step One

```
npm install -g react-native-cli
```
### Step Two

```
npm install
```
### Step Three

```
react-native start
```
### Step Four

```
react-native run-ios
react-native run-android
```
